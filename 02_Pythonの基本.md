# Python知識
- eで指数使える
  - 1.2e5で1.2*10^5
  - 1e-3で1*10^-3
- math.sqrt(n): 平方関数
- math.exp(n): 指数関数
- math.log(n): 自然対数関数
- math.pi: π
- math.cos(math.pi): 余弦
- math.sin(math.pi / 2): 余弦
- math.tan(math.pi): 余弦
- 改行は\を挟んで
  - ()内、リスト内などは省略可
